# CircleOne Keyboard -- Privacy Policy

**Effective date:** 2026-03-23
**Package name:** africa.one.keyboard
**Developer:** The Geek Network (Pty) Ltd

---

## Summary

CircleOne does not collect, store, transmit, or share any personal data. Your
keystrokes never leave your device.

---

## Data Collection

CircleOne collects **no data whatsoever**. Specifically:

- **No keystroke data** is recorded, logged, or transmitted.
- **No personal information** (name, email, phone number, location) is
  collected.
- **No usage analytics** or telemetry of any kind is embedded in the app.
- **No crash reports** that contain user-typed content are generated or sent.
- **No advertising SDKs** or ad networks are included.
- **No third-party tracking libraries** are bundled.
- **No network connections** are initiated by the keyboard. The app does not
  require internet permission.
- **No account creation** is required to use the keyboard.

## Learned Words and Personal Dictionary

CircleOne may offer a personal dictionary feature that learns words you type
frequently. This data is stored **exclusively on your device** in the app's
private storage. It is never transmitted, backed up to external servers, or
shared with any party.

If you uninstall CircleOne, all locally stored dictionary data is permanently
deleted by the Android operating system.

## Permissions

CircleOne requests only the permissions strictly necessary for a keyboard to
function:

- **Input Method permission** -- required by Android for any app that provides
  a software keyboard.
- **Vibration** (optional) -- for haptic feedback when keys are pressed.

No permission for internet access, contacts, microphone, camera, or storage is
requested.

## Third-Party Code

CircleOne is built on HeliBoard, which is built on OpenBoard, which is built on
the Android Open Source Project (AOSP) keyboard. All upstream code is open
source and subject to the same no-data-collection principle. No proprietary
third-party libraries are included.

## Children's Privacy

CircleOne does not collect data from any user, including children under 13. The
app is safe for users of all ages.

## Changes to This Policy

If this policy is updated, the revised version will be published at the same URL
and the effective date above will be updated. Because CircleOne collects no
data, material changes to this policy are unlikely.
