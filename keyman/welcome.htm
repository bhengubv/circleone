CircleOne -- isiBheqe soHlamvu Keyboard

Welcome, and thank you for installing the CircleOne keyboard. You can now type isiBheqe soHlamvu script directly from your keyboard using familiar Latin transliteration. The keyboard converts your keystrokes into the corresponding isiBheqe glyphs automatically.

How to use the keyboard

Input follows consonant-first (Latin) order. Type the Latin transliteration of a syllable and the keyboard produces the matching isiBheqe glyph. For example, type ba to produce the ba syllable glyph.

  • Pure vowels: a, e, i, o, u
  • Click consonants: c (dental), q (palatal), x (lateral)
  • Prenasalised: mb, nd, ng, nj, nk, nt
  • Aspirated: bh, kh, ph, th

Combine any consonant or consonant cluster with one of the five vowels to form a complete syllable. The keyboard handles the mapping for you.

The five core vowel shapes

Key Sound Shape Name
a /a/ Downward triangle Isoka
e /ɛ/ Leftward triangle Iphambili
i /i/ Upward triangle Intombi
o /ɔ/ Rightward triangle Imuva
u /u/ Upward chevron Umkhonto
Font required: The font one.ttf must be installed on your system for isiBheqe glyphs to render correctly. It should have been included with this keyboard package.
