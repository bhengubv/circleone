CircleOne -- isiBheqe soHlamvu Keyboard

About isiBheqe soHlamvu

isiBheqe soHlamvu is a featural syllabary designed for writing siNtu (Southern Bantu) languages. Unlike the Latin alphabet, each glyph in isiBheqe represents a complete syllable, and the shape of each glyph encodes phonetic information -- the vowel determines the overall geometric form while consonant features are marked systematically within it. This makes the script a natural fit for the syllable-timed phonology of languages such as isiZulu, isiXhosa, Sesotho, Setswana, SiSwati, Tshivenda, and Xitsonga.

The CircleOne keyboard lets you type isiBheqe script on any platform supported by Keyman. You type using familiar Latin transliteration and the keyboard converts your input into the corresponding isiBheqe glyphs in real time.

What will be installed

  • Keyboard layout: CircleOne input method for Keyman, mapping Latin keystrokes to isiBheqe syllable glyphs.
  • Font: one.ttf -- the isiBheqe soHlamvu display font, required for glyphs to render correctly.
PUA font note: The isiBheqe glyphs are mapped to the Unicode Private Use Area (PUA). This means the font one.ttf must be present on any system where the text is to be displayed. Without it, characters will appear as empty boxes or question marks.

System requirements

  • Keyman 14.0 or later
