# CircleOne Keyman Keyboard - Build Instructions

Step-by-step guide to building, testing, and submitting the CircleOne isiBheqe soHlamvu keyboard.

## Prerequisites

- **Keyman Developer 17.0+** -- free download from https://keyman.com/developer/
- **Windows** -- Keyman Developer is Windows-only; Linux/macOS users need WINE
- The font file **one.ttf** must exist in the `font/` directory (generated by FontForge)
- **Git**

## Building the Keyboard

### Step 1: Open the Project

1. Open Keyman Developer.
2. Go to **File > Open** and navigate to `keyman/circleone.kmn`.

### Step 2: Compile

1. Press **F7** or go to **Build > Compile**.
2. This generates:
   - `circleone.kmx` -- desktop keyboard
   - `circleone.js` -- web/mobile keyboard
3. Check the **Messages** pane for any errors.

### Step 3: Test Locally

1. In Keyman Developer, press **F5** to test.
2. Type Latin transliterations and verify isiBheqe glyphs appear.
3. Test cases:
   - `a` -> pure vowel /a/ glyph
   - `ba` -> ba syllable
   - `nka` -> prenasalised velar + a
   - `ca` -> dental click + a
   - `m` (alone) -> syllabic nasal

### Step 4: Build the Package

1. Open `circleone.kps` in Keyman Developer.
2. Verify all files are listed:
   - `circleone.kmn`
   - `one.ttf`
   - `welcome.htm`
   - `readme.htm`
3. Go to **Build > Compile Package**.
4. This generates `circleone.kmp` (the distributable package).

### Step 5: Test the Package

1. Double-click `circleone.kmp` to install in Keyman.
2. Verify:
   - Font installs correctly
   - Keyboard appears in Keyman
   - Input works in any text field
   - Welcome page displays after install

## Submitting to Keyman Repository

### Step 1: Fork keymanapp/keyboards

1. Go to https://github.com/keymanapp/keyboards
2. Click **Fork**.

### Step 2: Create submission branch

```bash
git clone https://github.com/YOUR_USERNAME/keyboards.git
cd keyboards
git checkout -b circleone
```

### Step 3: Copy files to submission structure

```bash
mkdir -p release/c/circleone/source
cp circleone.kmn release/c/circleone/source/
cp circleone.kps release/c/circleone/source/
cp circleone.keyman-touch-layout release/c/circleone/source/
cp one.ttf release/c/circleone/
cp welcome.htm release/c/circleone/
cp readme.htm release/c/circleone/
cp HISTORY.md release/c/circleone/
cp LICENSE.md release/c/circleone/
```

### Step 4: Build using the repository's build system

```bash
cd release/c/circleone
./build.sh
```

### Step 5: Submit PR

1. Push the branch and open a PR against `keymanapp/keyboards` master.
2. Title: **"Add CircleOne isiBheqe soHlamvu keyboard"**
3. The description should reference the script, supported languages, and link to https://isibheqe.org.za
4. The Keyman team will review against D.I.S.C.U.S. criteria.

## D.I.S.C.U.S. Quality Checklist

- **Documentation**: `welcome.htm` and `readme.htm` present and informative
- **Input**: All transliteration rules compile and produce correct output
- **Standards**: Uses Unicode PUA (acceptable for unencoded scripts)
- **Completeness**: Covers isiZulu core phoneme inventory (~220 syllables)
- **Usability**: QWERTY layout with intuitive long-press for complex consonants
- **Support**: `HISTORY.md`, `LICENSE.md`, GitHub repo linked
